情商大师 (EQ Master) 隐私政策

情商大师（以下简称“本应用”或“我们”）尊重并致力于保护您的隐私。我们将严格依照本隐私政策（以下简称“本政策”）收集、使用、保护您的个人信息。本政策适用于您在使用情商大师应用时收集的所有信息。请您在使用前仔细阅读并了解本政策，使用即表示您同意本政策的内容。

一、信息收集

为向您提供优质的情商提升服务，我们会根据“最小必要”原则收集您的信息。主要包括以下几类：1.

个人信息：您在注册时提供的个人信息，例如手机号、电子邮件、昵称等。我们会尽量避免收集可能导致敏感性或身份识别的信息，除非用于服务所必需。

2.

设备信息：当您使用本应用时，我们可能会自动收集您的设备信息，如设备型号、操作系统版本、设备标识符等，以帮助我们分析产品性能并优化体验。

3.

日志信息：为改进服务，我们会收集您在使用本应用时的操作日志，包括点击行为、浏览记录、故障日志等。

4.

位置信息：在您同意的情况下，我们可能收集您的位置信息，以便提供个性化服务。您可随时在设备设置中关闭位置权限。

二、信息使用

我们收集的信息将用于以下目的：1.

提供核心服务：用于创建账户、提供个性化训练内容、确保服务正常运行等。

2.

改进用户体验：分析您的行为数据，优化功能与服务，提高产品稳定性。

3.

信息推送：在获得您同意后，我们可能通过应用内消息、短信、电子邮件等方式向您发送与情商提升相关的内容、服务更新或活动通知，您可在应用内选择是否接收。

4.

数据安全：用于维护系统安全，防止恶意攻击、欺诈等非法行为。

三、信息分享与披露

我们承诺严格保密您的信息，未经您明确同意，不会向第三方披露或共享您的个人信息，除非：1.

获得您的授权：在您授权的情况下，依据授权范围使用您的信息。

2.

法律要求：在法律法规要求的情况下，我们可能会向相关司法、执法或其他有权机关提供您的信息。

3.

业务转让：在涉及本应用的合并、收购或资产转让时，信息可能会作为资产之一进行转移，我们会确保新的接收方继续遵守本政策。

四、信息安全

我们致力于保护您的信息安全，采取符合行业标准的安全措施，包括但不限于：1.

数据加密：通过传输加密、存储加密等技术手段，确保您的信息安全。

2.

访问控制：仅授权人员能够访问您的个人信息，并设有严格的权限控制措施。

3.

安全审查：我们会定期进行安全评估和数据安全审查，确保防护措施的有效性。

请注意，由于互联网的公开性，我们无法保证您的信息在所有情况下均绝对安全。因此，我们建议您妥善保管个人账号信息，不要轻易将密码透露给他人。

五、您的权利

您在使用情商大师时，享有以下权利：1.

访问和更正：您可在应用内查看和更正您的个人信息，确保信息准确、完整。

2.

删除：在法律允许的情况下，您有权请求我们删除您的部分或全部个人信息，您可在应用内联系我们提出请求。

3.

撤回授权：您可随时在设备权限设置中撤回我们对设备位置、通知等的访问权限。请注意，撤回授权可能影响部分服务的使用效果。

六、第三方服务

在某些情境下，本应用可能会引入第三方提供的服务（如支付、数据分析）。此类第三方可能会根据其隐私政策收集您的部分信息，情商大师不对其隐私行为负责。我们建议您在使用前查看该等第三方的隐私政策。

七、未成年人隐私保护

本应用主要面向成人用户，特别是25-35岁的用户人群。我们不会有意收集18岁以下未成年人的信息。如果我们发现未经监护人同意而收集了未成年人的信息，我们将尽快删除相关数据。

八、隐私政策的更新

  1. 政策变更：我们会不时更新本政策。若本政策有重大变更，我们将在应用内显著位置通知或向您发送通知。

  2. 继续使用：如果您在政策变更后继续使用本应用，即视为您接受更新后的政策。建议您定期查阅本政策，以了解最新内容。
